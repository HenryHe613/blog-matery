注册Valine并获取Appid和Appkey

Valine注册很奇怪，不是去官网注册，而是去LeanCloud注册。

[!TIP]

注意，免费版的Valine每天强制休眠6小时，也就是说，一天中有6小时无法评论。

Valine有两个版本，一个是国际版，还有一个是国内版。
国际版不需要实名认证，国内版需要实名认证。

我选择国际版hh

链接： 国际版 国内版

注册不用我多说。

点击创建应用,选择开发版，名字任意。

进入项目，选择 设置=>应用Keys=>Credentials

你可以看见AppID和AppKey。

Matery主题开启并配置Valine

打开主题文件的_config.yml，找到Valine，大概在303行左右。

# The configuration of the Valine comment module is not activated by default.
# To use it, activate the configuration item and set appId and appKey.
# Valine 评论模块的配置，默认为不激活，如要使用，就请激活该配置项，并设置 appId 和 appKey.
valine:
  enable: true
  appId: 
  appKey: 
  notify: false
  verify: true
  visitor: true
  avatar: 'mm' # Gravatar style : mm/identicon/monsterid/wavatar/retro/hide
  pageSize: 10
  placeholder: '来不及解释了，快上车~~~' # Comment Box placeholder
  background: /medias/comment_bg.png

把enalbe改成true，appId和appKey改成刚刚从leancloud获取的，

hexo cl&&hexo g&&hexo d看一下，应该就有Valine的评论区了。

[!WARNING]

yaml语法规定，:后面必须要有一个Space！！！不然会出错
